Installations


  • Tout d'abord, avant l'installation d'un paquet il est important et conseillé de faire un "apt update". Ensuite, pour installer Cmatrix, qui est donc un programme qui affiche un effet de matrice de code sur votre terminal, il faut faire la commande "apt install cmatrix"

    Une fois l'installation terminé, vous pouvez donc d'ors et déjà tester l'application.

  • Ensuite, pour installer Figlet, qui est un programme permettant de créer du texte ASCII, rien de très différent, vous pouvez effectuer la commande "apt install figlet"

    De nombreuses commandes sont disponibles, et si ce n'est pas déjà fait, je vous invite à retourner en arrière pour voir ce qu'il est possible de faire.



  • Les deux autres applications, Cowsay qui est un programme permettant de faire parler une vache et d'autres animaux en fonctions des commandes, puis SL qui est un programme affichant un train tout ça une fois de plus en ASCII sur votre terminal, sont des programmes amusants et n'ayant aucun but en particulier.

    Il est important de spécifier que ces programmes ont besoin de leurs chemins absolu pour être éxecutés. Pour cowsay il faudra donc faire la commande "apt install cowsay" et pour l'utiliser il faudra ensuite faire "/usr/games/cowsay [Puis la commande que vous voulez faire]."
    Pour SL (Steam Locomotive) c'est le même principe, "apt install sl" puis "/usr/games/sl [Puis la commande que vous voulez utiliser]"

Cmatrix

Défilement des caractères en différés: 

cmatrix -a


Activer le mode "gras" 

cmatrix -b


Utiliser une vitesse de défilement plus rapide pour l'animation. 

cmatrix -f


Utiliser la police d'écriture "matrix" sur la console 

cmatrix -l


Activer le mode "verrouillage" 

cmatrix -L


Utiliser un ancien style de défilement des caractères 

cmatrix -o


Afficher les commandes disponibles pour Cmatrix 

cmatrix -h


Activer le mode "Screensaver" (Appuyer sur une touche pour quitter) 

cmatrix -s


Afficher les informations à propos de la version de Cmatrix 

cmatrix -V


Contrôler la vitesse de défilement (de 0 à 10) 

cmatrix -u 10


Définir une couleur parmis les 8 disponibles 

cmatrix -C blue


Activer le mode arc-en-ciel 

cmatrix -r

Figlet

Utiliser une police de caractères spécifiés dans le dossier /usr/games/figlet 

figlet -f [nom de la police]


Centrer le texte horizontalement 

figlet -c


Spécifier la largeur de sortie du texte 

figlet -w 75


Activer les espaces entre les lettres 

figlet -k


Activer le mode STDIN permettant la saisie dans le terminal. 

figlet -p


Afficher le texte à droite 

figlet -r


Afficher le texte à gauche 

figlet -l

Cowsay

Spécifier une autre image ASCII parmis celles disponibles (voir plus d'infos) 

cowsay -f [nom du dessin]


Spécifier la largeur de la bulle de texte (40 par défaut) 

cowsay -w 80


Changer la direction de la bulle vers la droite 

cowsay -d right


Changer la direction de la bulle vers la haut 

cowsay -u


Changer la direction de la bulle vers la gauche 

cowsay -d left


Afficher la liste des dessin réalisable 

cowsay -d
